# Mini CRM System (HTML, CSS, JS)

## Features
- Admin login (username: admin, password: 1234)
- Add new leads (name, email, phone)
- Update lead status (new → contacted → converted)
- Add follow-up notes
- Data stored in browser localStorage

## Setup
1. Clone this repository.
2. Open `index.html` in your browser.
3. Login with admin credentials.
4. Manage leads via dashboard.

## Notes
This project simulates a CRM system using only HTML, CSS, and JavaScript.  
It demonstrates how businesses can track leads, update statuses, and manage follow-ups.
